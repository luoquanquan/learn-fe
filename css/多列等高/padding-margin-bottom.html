基于 padding-margin-bottom 实现多列等高

  • 利用 float 实现三列元素水平对齐
  • 父级 overflow: hidden; 清除浮动, 此时就能实现父级高度就是最高元素的高度
  • 每个列元素添加一个极大 padding 就能把父级撑得很高
  • 在给每个元素相同大小的负 marigin 还原布局
  • 由于 padding 可以填充元素得背景, 如果兄弟子元素高度大于当前元素, 看到的就是之前的 padding 区域. 由此来实现看上去多列等高的效果
