# pwa-vue

## 写在最前边

如果想要调试这个项目需要先把 `../server.js` 这个文件跑起来, 因为使用 pwa 缓存接口的时候要用到. 访问 `http://127.0.0.1:3333/pwa-vue-demo`

```shell
npm install
```
