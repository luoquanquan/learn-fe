## 2. [wagmi](https://github.com/wevm/wagmi)

**类型**: Web3 React Hooks | **语言**: TypeScript | **⭐**: 8.5k

### 简介

React Hooks 库，用于以太坊交互。

### 特点

- 🔌 20+ Hooks 覆盖常见场景
- 🎯 TypeScript 原生支持
- 🔗 多钱包支持（MetaMask、WalletConnect 等）
- ⚡ 自动缓存和刷新

### 核心 Hooks

```typescript
import { useAccount, useConnect, useDisconnect } from 'wagmi'

function Profile() {
  const { address, isConnected } = useAccount()
  const { connect } = useConnect()
  const { disconnect } = useDisconnect()

  if (isConnected)
    return (
      <div>
        Connected to {address}
        <button onClick={() => disconnect()}>Disconnect</button>
      </div>
    )
  return <button onClick={() => connect()}>Connect Wallet</button>
}
```

## 3. [langchain](https://github.com/langchain-ai/langchain)

**类型**: LLM 应用框架 | **语言**: Python/TS | **⭐**: 95.3k

### 简介

构建基于大语言模型应用的框架。

### 特点

- 🔗 链式调用 LLM
- 📚 RAG 检索增强生成
- 🛠️ 工具调用（Tools）
- 🧠 Agent 自动决策

### 快速开始

```python
from langchain import OpenAI, LLMChain, PromptTemplate

llm = OpenAI(temperature=0.9)
template = """你是一个程序员助手。

用户问题: {question}

请给出简洁的回答:"""

prompt = PromptTemplate(template=template, input_variables=["question"])
chain = LLMChain(llm=llm, prompt=prompt)

chain.predict(question="什么是 React Hooks?")
```

## 5. [tinygrad](https://github.com/tinygrad/tinygrad)

**类型**: 深度学习框架 | **语言**: Python | **⭐**: 26.1k

### 简介

一个极简的深度学习框架，适合学习理解深度学习原理。

### 特点

- 🎯 代码简单易懂（核心代码 <1000 行）
- 🚀 支持 GPU 加速
- 🔧 支持多种后端（CPU、GPU、TPU）
- 📚 优秀的学习资源

### 示例

```python
from tinygrad import Tensor, nn

class Model:
  def __init__(self):
    self.l1 = nn.Linear(784, 128)
    self.l2 = nn.Linear(128, 10)
  
  def __call__(self, x):
    x = self.l1(x).relu()
    return self.l2(x)

model = Model()
optim = nn.optim.Adam(nn.state.get_parameters(model))

# 训练循环
for i in range(100):
  loss = model(X).sparse_categorical_crossentropy(Y).backward()
  optim.step()
```
