## 重点方向

### 前端方向侧重
1. 移动端开发（React Native / Flutter）
2. WebView 与 Native 通信
3. 性能优化（虚拟列表、内存管理）
4. TypeScript 工程实践
5. 安全存储（Keychain/Keystore）

### Web3 方向侧重
1. 钱包连接标准（EIP-1193、EIP-6963、WalletConnect）
2. 私钥管理和安全
3. 多链支持
4. 交易签名和状态追踪
5. 常见攻击防护（钓鱼、假币、授权风险）
6. DeFi 集成（Swap、Staking）

### AI 方向侧重
1. AI + Web3 安全审计
2. RAG 在 DApp 中的应用
3. AI Agent 自动化交易
4. 智能客服和交易解释
