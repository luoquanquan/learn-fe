## 1. [vercel/ai](https://github.com/vercel/ai)

**类型**: AI SDK | **语言**: TypeScript | **⭐**: 12.5k

### 简介

Vercel 出品的 AI SDK，用于构建流式 AI 应用的 React/Vue/Svelte SDK。

### 特点

- 🌊 内置流式响应支持
- 🔄 自动管理 AI 状态
- 🔌 支持 OpenAI、Anthropic、Hugging Face
- 📝 内置聊天界面组件

### 快速开始

```bash
npm install ai openai
```

```typescript
import { OpenAIStream, StreamingTextResponse } from 'ai'
import OpenAI from 'openai'

const openai = new OpenAI()

export async function POST(req: Request) {
  const { messages } = await req.json()
  
  const response = await openai.chat.completions.create({
    model: 'gpt-4',
    stream: true,
    messages,
  })
  
  const stream = OpenAIStream(response)
  return new StreamingTextResponse(stream)
}
```

## 4. [drizzle-team/drizzle-orm](https://github.com/drizzle-team/drizzle-orm)

**类型**: ORM | **语言**: TypeScript | **⭐**: 23.1k

### 简介

类型安全的 TypeScript ORM，SQL-like 语法。

### 特点

- 🎯 类型安全
- 📝 SQL-like 语法
- 🔍 自动推导类型
- 📦 轻量级

### 示例

```typescript
import { drizzle } from 'drizzle-orm/node-postgres'
import { eq } from 'drizzle-orm'
import { users } from './schema'

const db = drizzle(client)

// 查询
const result = await db.select().from(users).where(eq(users.id, 1))

// 插入
await db.insert(users).values({ name: 'John', email: 'john@example.com' })
```

## 5. [honojs/hono](https://github.com/honojs/hono)

**类型**: Web 框架 | **语言**: TypeScript | **⭐**: 18.9k

### 简介

超快的、轻量级的、基于 Web Standards 的 Web 框架。

### 特点

- ⚡ 超快路由
- 🪶 轻量级（<20KB）
- 🌐 多运行时支持（Cloudflare Workers、Deno、Bun、Node.js）
- 📝 TypeScript 原生

### 示例

```typescript
import { Hono } from 'hono'

const app = new Hono()

app.get('/', (c) => c.text('Hello Hono!'))

app.get('/api/users/:id', (c) => {
  const id = c.req.param('id')
  return c.json({ id, name: 'John' })
})

export default app
```
