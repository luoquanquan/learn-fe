## з”ҹжҲҗдёҖдёӘ [min, max] зҡ„йҡҸжңәж•ҙж•°

- з”ҹжҲҗйҡҸжңәж•°иӮҜе®ҡиҰҒз”ЁеҲ° `Math.random() -> [0, 1)`
- еҸҜд»ҘжҠҠдёҖдёӘж•°еҸҳжҲҗж•ҙж•°зҡ„ж–№жі•жңүпјҡ`parseInt() Math.floor() Math.round() Math.ceil()`
- иҝҷйҮҢйҖүжӢ©еҠҹиғҪејәеҠІ([дёәд»Җд№Ҳиҝҷд№ҲиҜҙе‘ў](https://luoquanquan.github.io/note/2020/02/04/basic-js/parseInt-vs-Math-floor))зҡ„ parseInt зӣҙжҺҘжҲӘеҸ–ж•°еӯ—зҡ„ж•ҙж•°йғЁеҲҶеҗ‘дёӢеҸ–ж•ҙ
- йӮЈд№ҲдёҚйҡҫеҫ—еҲ° `parseInt(Math.random() * 10) -> [0, 9]`
- жүҖд»ҘпјҢеҰӮжһңжҲ‘д»¬жғіиҰҒеҫ—еҲ°дёҖдёӘ [0, max] зҡ„йҡҸжңәж•°еҸӘйңҖиҰҒ `parseInt(Math.random() * (max + 1))`
- жғіиҰҒеҫ—еҲ°дёҖдёӘ [1, max] зҡ„йҡҸжңәж•°еҸӘйңҖиҰҒ `parseInt(Math.random() * (max - 1 + 1)) + 1`
- иҝӣиҖҢеҸҜеҫ—еҮәпјҢжғіиҰҒеҫ—еҲ°дёҖдёӘ [min, max] зҡ„йҡҸжңәж•°гҖӮжҠҠжӢ¬еҸ·йҮҢзҡ„ -1 е’ҢжӢ¬еҸ·еӨ–зҡ„ +1 зӯүйҮҸд»ЈжҚўжҲҗ min `parseInt(Math.random() * (max - min + 1)) + min`

жңҖз»ҲеҸҜд»Ҙеҫ—еҮәпјҢз”ҹжҲҗдёҖдёӘ [min, max] зҡ„йҡҸжңәж•ҙж•°зҡ„д»Јз Ғдёә
```js
const randomNum = (min = 0, max = min) => {
  if (max === min) {
    return parseInt(Math.random() * (min + 1))
  } else {
    return parseInt(Math.random() * (max - min + 1) + min)
  }
}
```

let i = 0
let ret = []
const randomNum = (min, max) => {
    const range = max - min
    const random = Math.random()
    // еӣӣиҲҚдә”е…Ҙ, еӯҳеңЁжңҖе°ҸеҖје’ҢжңҖеӨ§еҖјжҰӮзҺҮдёәдёҖеҚҠзҡ„й—®йўҳ
    return min + Math.floor(random * range)
  }
while(i++ < 1e7) {
  const random = randomNum(1, 10)
  if (ret[random]) {
    ret[random]++
  } else {
    ret[random] = 1
  }
}

ret = ret.map(i => [i, `${(i / 1e7 * 100).toFixed(2)}%`])
