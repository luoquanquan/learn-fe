# 17. 打印从 1 到最大的 n 位数 - 解决方案

## 题目描述

输入数字 n, 按顺序打印出从 1 到最大的 n 位十进制数。比如输入 3 则依次打印 1,2,3⋯⋯999.

示例 1:
```
输入: n = 1
输出: [1,2,3,4,5,6,7,8,9]
```

## 解题方案

首先求出要打印的数字范围，再从 1 开始打印到最大的数字

## 算法流程

这个比较简单，n 位数其实就是 10 的 n 次方。直接一个 for 循环搞定

## 代码

```js
const printNumbers = n => {
    const max = 10 ** n
    const ret = []
    for (let i = 1; i < max; i++) {
        ret.push(i)
    }

    return ret
}
```
