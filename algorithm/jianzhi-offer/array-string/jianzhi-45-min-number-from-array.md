# 45. 把数组排成最小的数

## 题目描述

输入一个非负整数数组，把数组里所有数字拼接起来排成一个数，打印能拼接出的所有数字中最小的一个。

示例 1:
```js
输入: [10, 2]
输出: '102'
```

示例 2:
```js
输入: [3, 30, 34, 5, 9]
输出: '3033459'
```

提示：`0 < nums.length <= 1000`

## 解题思路

- 拼接数组内所有元素使结果最小，本质上是排序
- 若字符串拼接 a + b > b + a, 那么应该让 b < a(b 在 a 的左边)
- 根据上述规则对数组中的所有元素排序

## 代码

```js
const minNumber = nums => {
    const strs = []

    // 数字元素转字符串后存储
    nums.forEach(ele => strs.push('' + ele))

    strs.sort((a, b) => (a + b) - (b + a))

    return strs.join('')
}

minNumber([3, 30, 34, 5, 9])
```
